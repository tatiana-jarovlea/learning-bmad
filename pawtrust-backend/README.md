# PawTrust Backend

Laravel API backend for the PawTrust platform.

## Prerequisites

- PHP 8.2+
- Composer 2.x
- Docker (for MySQL + Redis)

## Local Setup

```bash
# 1. Clone and install dependencies
composer install

# 2. Copy environment file and generate app key
cp .env.example .env
php artisan key:generate

# 3. Start MySQL + Redis via Docker
docker compose up -d

# 4. Run database migrations
php artisan migrate

# 5. Start dev server on port 8000
php artisan serve
```

## Environment Variables

Copy `.env.example` to `.env` and fill in:

| Variable | Description |
|---|---|
| `APP_KEY` | Generated by `php artisan key:generate` |
| `DB_*` | MySQL connection (Docker defaults work out of the box) |
| `AWS_ACCESS_KEY_ID` / `AWS_SECRET_ACCESS_KEY` | AWS credentials for S3 + SES |
| `AWS_BUCKET` | S3 bucket name for private file storage |
| `AWS_CLOUDFRONT_URL` | CloudFront distribution URL for photo delivery |
| `STRIPE_KEY` / `STRIPE_SECRET` / `STRIPE_WEBHOOK_SECRET` | Stripe API keys |
| `SKIP_LISTING_PAYMENT` | Set `true` locally to bypass Stripe payment requirement |
| `ADMIN_EMAIL` / `ADMIN_PASSWORD` | Credentials for initial admin seeder |
