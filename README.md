Welcome to the BMAD learning project made by @tatiana-jarovlea.
This repository is used to learn and understand basics of BMAD framework.


Plan for BMAD learning project:
1. Clone this project
2. Open in the VS code
3. Install BMAD framework 4.44.1 version using command `npx bmad-method@4.44.1 install`.
4. Choose Claude agent (preferably)

Here is a complete BMAD V4 user guide: https://github.com/bmad-code-org/BMAD-METHOD/blob/V4/docs/user-guide.md
Good luck! :)
